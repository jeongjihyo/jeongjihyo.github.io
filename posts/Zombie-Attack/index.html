ZOMBIE ATTACK 게임 시작 화면



Unity Asset Store에 등록되어 있는 3D 모델을 다운받아 좀비와 1인칭 사람에 움직일 수 있도록 애니메이션을 적용했습니다. 좀비는 자동적으로 움직일 수 있도록 설정해 놓았으며 1인칭인 사람은 키보드 A(오른쪽), D(왼쪽), W(직진), S(후진)으로 움직일 수 있습니다. 총과 총알을 다운받아 C#으로 마우스의 왼쪽을 누르면 총알이 나갈 수 있도록 구현했습니다.

[ Game Strart ]버튼을 누르면 3가지 총 디자인 중 본인이
원하는 총을 선택하여 좀비와 싸울 수 있습니다.



좀비를 3명 이상을 죽여야 게임을 성공하며 죽일 때마다 점수가 올라 총 60점 맞아야 게임에서 이기게 됩니다.

좀비에게 닿으면 HP가 깎이며 시작 100HP에서 0HP가
되면 죽게됩니다.

오른쪽 하단의 숫자 / 무한의 표시는 총알의 갯수를 표현한 것 입니다.
