쇼핑몰 DumbDumb Main 화면



DumbDumb은 개인쇼핑몰로 사용자용과 관리자용으로 나뉘어져 있습니다. 사용자는 주문 · 결제 · 1:1문의 등이 가능하며 관리자는 회원관리 · 상품관리 · 주문관리 · 옵션관리 · FAQ관리를 할 수 있습니다.


[ 사용자용 화면 ]

DumbDumb의 사용자용 메인화면으로
홈으로 가기 / 로그인 / 회원가입 / 장바구니 / 주문 / 즐겨찾기 메뉴가
상단에 있으며 왼쪽은 상품의 카테고리로 분류되어 있습니다.


회원가입을 하고 로그인 후에 주문 / 결제 등 이용이 가능합니다.


장바구니에 담겨져 있는 상품을 주문 시 카드정보 입력 후
[ 결제하기 ] 버튼을 누르면 결제가 진행됩니다.


결제 이후 주문내역 / 배송내역 / 결제내역을 확인 할 수 있습니다.


[ 관리자용 화면 ]

DumdDumd의 관리자용 화면으로
회원관리 / 상품관리 / 주문관리 / 옵션관리 / FAQ관리 탭으로 되어 있습니다.
상단에 있으며 왼쪽은 상품의 카테고리로 분류되어 있습니다.

회원관리 탭에서는 회원의 아이디 · 이름 · 휴대폰 번호 · 이메일을
확인할 수 있습니다.


상품관리 화면으로 상품의 등록 / 수정 / 삭제를 할 수 있으며
상품 수량 · 상품 분류 · 상품 코드 · 상품명 · 가격 · 상태 · 이벤트 등을
확인 할 수 있습니다.


해당 상품을 누르면

해당 상품의 변경 사항을 수정할 수 있습니다.


주문관리 화면으로 주문한 상품에 대해 확인을 할 수 있으며
주문번호 · 주문일 · 상품명 · 제품수량 · 주문자 · 결제 방법 · 주문 상태를
확인 할 수 있습니다.


해당 주문건을 누르면

주문한 고객님의 정보와 상품을 상세하게 조회할 수 있습니다.
